🟩 POLÍTICA DE PRIVACIDAD — InvoiceRapid PRO Última actualización: 02/05/2026 InvoiceRapid PRO (“la Aplicación”), desarrollada por ZKR Studio, se compromete a proteger la privacidad de los usuarios. Esta Política de Privacidad explica qué datos se recopilan, cómo se utilizan y qué derechos tiene el usuario. 1. Datos que recopilamos 1.1. Datos proporcionados por el usuario Correo electrónico y contraseña para crear una cuenta. Datos introducidos manualmente en la aplicación: clientes, productos, facturas y configuraciones. 1.2. Datos técnicos Tipo de dispositivo Sistema operativo Información básica de uso para mejorar la aplicación 1.3. Servicios externos La aplicación utiliza Supabase como servicio de autenticación y base de datos. 2. Uso de los datos Los datos se utilizan para: Crear y gestionar la cuenta del usuario Guardar y sincronizar facturas Mejorar la estabilidad y funcionamiento de la aplicación 3. Compartición de datos InvoiceRapid PRO no vende datos personales. Los datos solo se comparten con: Supabase, para autenticación y almacenamiento Autoridades legales si fuera requerido 4. Seguridad Implementamos medidas para proteger los datos: Conexiones cifradas (HTTPS) Acceso restringido mediante autenticación Almacenamiento seguro en Supabase 5. Derechos del usuario El usuario puede: Acceder a sus datos Rectificar información incorrecta Solicitar la eliminación de su cuenta y datos enviando un correo a: 📧 zkrstudio.contact@gmail.com 6. Eliminación de datos Cuando el usuario solicite la eliminación de su cuenta, sus datos serán eliminados manualmente de Supabase. 7. Cambios en esta política Podemos actualizar esta política ocasionalmente. La fecha de la última actualización se mostrará arriba. 8. Contacto Para dudas sobre esta política: 📧 zkrstudio.contact@gmail.com
